# LinguaVault

## Quick Start

### 1. Backend Setup
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Or use the run script:
- Windows: Double-click `run.bat`
- Linux/Mac: Run `./run.sh`

## Features

- Audio upload with drag & drop
- AI-powered transcription (Whisper)
- Automatic translation (GPT)
- Semantic phrase search
- Chat interface
- Archive with filters
- Analytics dashboard
- Endangered language support

## Tech Stack

- Frontend: React, Vite, Tailwind, Framer Motion
- Backend: Flask, SQLAlchemy
- AI: Whisper, OpenAI GPT, Sentence Transformers
- Database: SQLite
